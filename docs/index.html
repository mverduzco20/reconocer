ORIENTACIÓN URBANA DESDE EL RECORRIDO COTIDIANO

(Re)conocer

Un mapa colectivo para recorrer el territorio por medio de la observación y el juego.

¿Qué es?

Este proyecto busca crear un archivo vivo de hitos urbanos: lugares que las personas reconocen, recuerdan y usan para orientarse. A partir de San Ángel, Ciudad de México, se invita a explorar la ciudad por categorías — memoria, oficio, refugio y encuentro — y a sumar nuevas miradas al mapa.

Cómo funciona

  1. Elige un punto de partida en la cartografía.
  2. Filtra las categorías para definir tu ruta.
  3. Sube un hito: captura el momento y comparte una imagen.
  4. Descubre el archivo colectivo de la comunidad.
